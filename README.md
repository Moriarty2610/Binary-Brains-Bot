# Binary-Brains-Bot

### Install
```bash
$ npm i
```
- Add `.env` file

### Deploy Bot 
```bash
# run only once
$ node deploy-commands.js 
# Alternatively
$ npm run build
```

### Run Bot
```bash
$ npm run dev
```
